# **App Name**: FaceWeave

## Core Features:

- Image Upload: Image Upload: Allow users to upload a reference image and a face image.
- Image Fusion: Image Fusion: AI tool for fusing the face from the face image into the reference image.
- Image Display: Display the original reference image, the face image, and the fused image.
- Download Image: Provide a download button for users to save the fused image.
- Intuitive Design: Offer a simple and intuitive interface with clear instructions.

## Style Guidelines:

- Primary color: Soft Lavender (#D0BFFF) for a calming and creative feel.
- Background color: Light gray (#F0F0F5) for a clean and neutral backdrop.
- Accent color: Muted Blue (#A0B4E0) for subtle highlights and interactive elements.
- Headline font: 'Belleza' serif for headlines and short descriptions, creating an elegant impression. Body font: 'Alegreya' serif for longer descriptive text
- Use simple, modern icons for upload, download, and other actions.
- Arrange the image upload and display sections in a clear, sequential order for ease of use.
- Implement subtle animations for loading states and button interactions.
